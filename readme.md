##  Descrição do Projeto

Este projeto consiste em um **labirinto interativo gerado dinamicamente**, desenvolvido com **JavaScript puro, HTML e CSS**.  
O objetivo é criar automaticamente um labirinto válido, exibindo visualmente suas paredes e permitindo que o jogador se mova pelo cenário utilizando as setas do teclado.

O labirinto é gerado em tempo real por um algoritmo de geração procedural, garantindo que cada execução produza um caminho diferente, sempre com uma posição inicial e um ponto final bem definidos.

Além disso, o projeto conta com:

- geração automática da grade do labirinto;
- controle do jogador por teclado (setas direcionais);
- sistema de paredes que bloqueiam movimentos inválidos;
- posição de início e fim destacadas visualmente;
- contador de tempo para acompanhar a partida.

Durante o desenvolvimento, o foco foi criar uma base simples, organizada e fácil de entender, servindo tanto como um mini-jogo funcional quanto como um projeto de estudo para lógica, manipulação do DOM e estruturação de código em JavaScript.
